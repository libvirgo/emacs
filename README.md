# Emacs config

## Install

```shell
git clone git@github.com:libvirgo/emacs.git --recurse-submodules
git clone --depth 1 https://github.com/emacsmirror/epkgs.git .local/lib/epkgs
```

And move compiled librime to .local/lib/librime directory.
